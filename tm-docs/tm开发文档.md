# 目录结构
``` pyton

│  └─app        #应用
│      ├─controllers    #Control层负责控制逻辑处理

│      └─routers        #router层负责页面路径控制
│          └─v1
├─tm-docs       #开发文档
├─tm-frontend   #前端
│  ├─public     #公共资源
│  └─src        #前端源码

│      └─components #组件
├─tm-guide      #用户手册
├─tutorial      #教程
```
